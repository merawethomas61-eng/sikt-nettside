# Bulletproof React Architecture Følg arkitekturen fra [https://github.com/alan2207/bulletproof-react](https://github.com/alan2207/bulletproof-react). ## Mappestruktur: - /src/components: Felles UI-komponenter. - /src/features/[feature-name]/: - api/: API-kall. - components/: Feature-spesifikke komponenter. - types/: TypeScript-typer. - index.ts: Public API. ## Regler: 1. Ingen 'any' i TypeScript. 2. Forretningslogikk skal ligge i features, ikke i globale komponenter. 3. Bruk index.ts for å kontrollere eksponering.
